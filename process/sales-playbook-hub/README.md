## Usage

1. Start at the **Dashboard** to get an overview of all modules
2. Begin with **Training & Onboarding** for new team members
3. Use **Sales Playbooks** to access qualification frameworks
4. Navigate between tools using the navigation buttons
5. Progress auto-saves in browser local storage
6. Export reports as needed
